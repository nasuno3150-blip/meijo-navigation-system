# 名城大学 屋内ナビゲーションシステム
## Meijo University Indoor Navigation System

Web上で動く屋内用ナビゲーションシステムです。校内の各教室や施設への最適なルートを検索・表示します。

### 主な機能

✨ **QRコード出発地点読み込み**
- ブラウザのカメラを使用してQRコードをスキャン
- 出発地点を自動設定

📍 **地図上でのクリック選択**
- 地図をクリックして目的地を選択
- ドラッグ&ドロップでの選択にも対応予定

🛗 **複数ルート検索**
- エスカレーター優先ルート
- 階段優先ルート
- 異なるルートを比較表示

🗺️ **高解像度地図表示**
- 1階・2階の詳細な校内地図
- ノード・エッジベースのナビゲーション

🌐 **多言語対応**
- 日本語
- 英語

📝 **詳細なルート案内**
- ターン・バイ・ターン式の文字指示
- 地図上にルートを視覚的に表示
- 距離と所要時間を表示

### 技術仕様

**使用技術:**
- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API (地図描画)
- WebRTC API (カメラアクセス)
- jsQR (QRコード読み込み)

**ルート検索アルゴリズム:**
- Dijkstraアルゴリズムを使用した最短ルート検索

**座標システム:**
- 各ノードは (x, y) 座標とフロア情報を保有
- エッジはノード間の接続と距離情報を保有
- ウェイポイントは移動経路の中間地点として機能

### デザイン

**カラースキーム:**
- プライマリカラー: `#dc143c` (Crimson Red - 赤)
- セカンダリカラー: `#ffffff` (White - 白)
- テキストカラー: `#333333` (Dark Gray - 濃いグレー)

**レスポンシブ対応:**
- デスクトップ: 左サイドバー + 右地図
- タブレット: 上部操作パネル + 下部地図
- モバイル: フルスクリーン対応

### 使用方法

#### 基本的な使い方

1. **言語を選択**
   - 日本語または英語を選択

2. **出発地点を設定**
   - 「カメラで読み込み」ボタンをクリック
   - QRコードをカメラに向かせてスキャン
   - または直接テキスト入力

3. **目的地を選択**
   - リストから選択、または
   - 検索ボックスに入力して検索、または
   - 地図上をクリック

4. **オプションを設定**
   - エスカレーター優先 / 階段優先を選択

5. **ルート検索**
   - 「ルート検索」ボタンをクリック
   - 複数のルートが表示される場合は選択

6. **ルート情報を確認**
   - 距離と所要時間を確認
   - ターン・バイ・ターン指示に従う
   - 地図上のルートを確認

### QRコード生成方法

QRコードはオンラインツールを使用して生成できます：
- https://www.qr-code-generator.com/
- https://qr.quel.jp/

ノード名（例: `1C`, `2B`, `library`など）をQRコードの内容として設定してください。

### ノード一覧

#### 1階
- **教室:** 1C, 1B, 1A, 1D, 1E, 1F, 1G, 1H, 1I
- **施設:** 図書室, 保険室, 職員室
- **その他:** 噴水, エスカレーター, 階段, 玄関出入口

#### 2階
- **教室:** 2C, 2B, 2A, 2D, 2E, 2F, 2G, 2H, 2I, 3A, 3B, 3C, 3D, 3E, 3F, 3G, 3H, 3I
- **施設:** 指導室, 校長室, 倉庫, 空き教室
- **その他:** エスカレーター, 階段

### カスタマイズ

#### ノードの追加

`mapData.js`のNODESオブジェクトに以下の形式で追加：

```javascript
'新しい場所': {
    name: '新しい場所',
    floor: 1,           // 階数
    x: 200,            // X座標
    y: 300,            // Y座標
    type: 'classroom', // タイプ（classroom, facility, escalator, stairs, waypoint, entrance）
    nameJa: '新しい場所',
    nameEn: 'New Location'
}
```

#### エッジの追加

`mapData.js`のEDGES配列に以下の形式で追加：

```javascript
{ from: 'ノードA', to: 'ノードB', distance: 100, type: 'normal' }
```

#### 地図の拡張

複数階対応、別棟対応など機能拡張が可能です。

### トラブルシューティング

**カメラへのアクセスが拒否される場合**
- ブラウザの設定を確認
- https接続を使用してください
- プライベートブラウジングモードを無効にしてください

**QRコードが読み込めない場合**
- QRコードが十分に明るく、鮮明に映っているか確認
- カメラのフォーカスを調整
- 別のQRコード生成ツールを試す

**ルートが見つからない場合**
- ノードとエッジが正しく定義されているか確認
- 出発地点と目的地が正しく選択されているか確認
